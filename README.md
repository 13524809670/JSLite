## 核心Core
```js
$$("#box")              //选择器
$$.isFunction           //判断对象是否为【函数】
$$.isObject             //判断是否为【Object】
$$.isArray              //判断对象是否为【数组】
$$.likeArray            
$$.type                 //获取类型
$$.error                //创建自定义错误

$$("#box").each()       //获取类型
$$("#box").extend()
```
